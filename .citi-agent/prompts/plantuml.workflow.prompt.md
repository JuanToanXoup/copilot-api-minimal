Prompt:                                                                                                                                                                                                                                                  
                                                                                                                                                                                                                                                       
Create a PlantUML activity diagram for a workflow. The diagram must follow this schema:                                                                                                                                                                  
                                                                                                                                                                                                                                                       
## Structure                                                                                                                                                                                                                                             
- Start with `@startuml` and end with `@enduml`                                                                                                                                                                                                          
- Use `title Workflow Name` for the workflow name                                                                                                                                                                                                        
- Use `start` and `stop` for entry/exit points                                                                                                                                                                                                           
- Activities use `:Activity Name;` syntax                                                                                                                                                                                                                
                                                                                                                                                                                                                                                       
## Swimlanes (for agent assignment)                                                                                                                                                                                                                      
Define agents with: `|#Color|AgentName|`                                                                                                                                                                                                                 
Example: `|#LightBlue|Analyzer|`                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                       
## Activity Annotations                                                                                                                                                                                                                                  
Add notes after activities to configure them:                                                                                                                                                                                                            
                                                                                                                                                                                                                                                       
### Prompt Block (AI task):                                                                                                                                                                                                                              
```plantuml                                                                                                                                                                                                                                              
:Task Name;                                                                                                                                                                                                                                              
note right                                                                                                                                                                                                                                               
<<prompt>>                                                                                                                                                                                                                                             
template: template-id                                                                                                                                                                                                                                  
input: {{variableName}}                                                                                                                                                                                                                                
end note                                                                                                                                                                                                                                                 
                                                                                                                                                                                                                                                       
HTTP Request:                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                       
:Fetch Data;                                                                                                                                                                                                                                             
note right                                                                                                                                                                                                                                               
<<http>>                                                                                                                                                                                                                                               
method: GET                                                                                                                                                                                                                                            
url: https://api.example.com/endpoint                                                                                                                                                                                                                  
end note                                                                                                                                                                                                                                                 
                                                                                                                                                                                                                                                       
Aggregator (combine parallel results):                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                       
:Combine Results;                                                                                                                                                                                                                                        
note right                                                                                                                                                                                                                                               
<<aggregator>>                                                                                                                                                                                                                                         
strategy: merge                                                                                                                                                                                                                                        
end note                                                                                                                                                                                                                                                 
                                                                                                                                                                                                                                                       
Control Flow                                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                       
- Conditions: if (condition?) then (yes) ... else (no) ... endif                                                                                                                                                                                         
- Parallel: fork ... fork again ... end fork                                                                                                                                                                                                             
- Loops: repeat ... repeat while (condition?)                                                                                                                                                                                                            
                                                                                                                                                                                                                                                       
Variables                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                       
- {{input}} - workflow input                                                                                                                                                                                                                             
- {{variableName}} - output from upstream node                                                                                                                                                                                                           
                                                                                                                                                                                                                                                       
Example:                                                                                                                                                                                                                                                 
                                                                                                                                                                                                                                                       
@startuml                                                                                                                                                                                                                                                
title Document Processor                                                                                                                                                                                                                                 
                                                                                                                                                                                                                                                       
|#LightBlue|Extractor|                                                                                                                                                                                                                                   
|#LightGreen|Analyzer|                                                                                                                                                                                                                                   
|#Gold|Writer|                                                                                                                                                                                                                                           
                                                                                                                                                                                                                                                       
start                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                       
|Extractor|                                                                                                                                                                                                                                              
:Extract Text;                                                                                                                                                                                                                                           
note right                                                                                                                                                                                                                                               
<<prompt>>                                                                                                                                                                                                                                             
template: text-extraction                                                                                                                                                                                                                              
input: {{input}}                                                                                                                                                                                                                                       
end note                                                                                                                                                                                                                                                 
                                                                                                                                                                                                                                                       
|Analyzer|                                                                                                                                                                                                                                               
:Analyze Content;                                                                                                                                                                                                                                        
note right                                                                                                                                                                                                                                               
<<prompt>>                                                                                                                                                                                                                                             
template: content-analysis                                                                                                                                                                                                                             
input: {{extractedText}}                                                                                                                                                                                                                               
end note                                                                                                                                                                                                                                                 
                                                                                                                                                                                                                                                       
if (needs review?) then (yes)                                                                                                                                                                                                                            
:Flag for Review;                                                                                                                                                                                                                                      
else (no)                                                                                                                                                                                                                                                
|Writer|                                                                                                                                                                                                                                               
:Generate Summary;

  <<prompt>>                                                                                                                                                                                                                                           
  template: summary-writer                                                                                                                                                                                                                             
  input: {{analysis}}                                                                                                                                                                                                                                  
end note                                                                                                                                                                                                                                               
endif                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                       
stop                                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                       
@enduml
